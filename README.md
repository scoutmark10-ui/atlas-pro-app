# 🚀 Atlas Pro Dashboard

Dashboard profissional para acompanhamento de métricas e metas pessoais.

## 📱 Funcionalidades

- 📊 7 métricas principais
- 🎯 Sistema de metas
- 🌓 Tema Dark/Light
- 📈 Gráfico radar interativo
- 💾 Salvamento automático

## 🔗 Acesse

[https://scoutmark10-ui.github.io/atlas-pro-app/index.html]

## 📲 Instalar como app

1. Abra o link no Chrome
2. Toque em ⋮ → "Adicionar à tela inicial"
3. Pronto! Ícone na área de trabalho

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript
- Chart.js
- PWA
